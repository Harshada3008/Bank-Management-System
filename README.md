# Real Time Bank Management System

A simple bank management system built with Python and SQLite.

## Features

- User registration and login
- Account management
- Deposits, withdrawals, transfers
- Transaction history
- Real-time balance updates (using SQL transactions for atomicity)

## Installation

1. Clone or download the project.
2. Install dependencies: `pip install -r requirements.txt`

## Usage

Run `python main.py` to start the CLI interface.

## Flask Web UI

Run `python app.py` and open `http://127.0.0.1:5000` in your browser.

## Testing

Run `pytest` in the root directory.
